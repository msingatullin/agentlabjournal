# Контентная методика

Этот документ задаёт способ работы, а не тему для публикации.

1. Выбирать сильные примеры в нужной нише.
2. Извлекать абстрактную структуру: хук, обещание, доказательство, шаги, CTA.
3. Не копировать формулировки, личные истории, цифры и рекламные обещания источника.
4. Переносить структуру на собственную тему, факты и опыт Agent Lab Journal.
5. Делать два продукта из одной темы: карусель до 8 слайдов и короткий Reels.
6. Каждый материал должен вести к конкретной статье или форме заявки.
7. Проверять факты и ссылку перед публикацией.

## Приоритеты

- Польза и конкретика вместо общих обещаний.
- Оригинальность вместо пересказа.
- Реальная воронка вместо набора подписчиков без следующего действия.
- Собственные подтверждённые метрики вместо неподтверждённых цифр из чужих кейсов.
